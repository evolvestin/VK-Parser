# VK Community Parser (VK to Telegram)

## Описание проекта
Программное обеспечение для автоматизированного сбора (парсинга) публикаций со стен сообществ социальной сети «ВКонтакте» (VK.com) и их последующей трансляции (репостинга) в каналы или группы мессенджера Telegram. Решение разработано на базе официальной документации VK API.

Основное назначение — создание зеркала контента сообщества VK в среде Telegram в автоматическом режиме.

## Функциональный алгоритм
1.  **Сбор данных:** Система запрашивает список последних 100 публикаций целевого сообщества VK.
2.  **Верификация:** Производится сверка уникальных идентификаторов (ссылок) постов с базой данных (реализована через Google Sheets) для исключения дублирования контента.
3.  **Обработка и публикация:** Тип контента идентифицируется, форматируется и отправляется в целевой ресурс Telegram.

## Технические характеристики и совместимость
Система поддерживает обработку следующих типов медиаконтента:
*   Текстовые публикации.
*   Изображения.
*   Видеоматериалы (нативная загрузка VK и ссылки на YouTube).
*   Составные (мультимедийные) посты.

### Известные ограничения
*   **Аудиоматериалы:** Обработка музыкальных треков не производится ввиду ограничений API и политики лицензирования контента «ВКонтакте».
*   **Нативное видео VK:** Загрузка видеофайлов, размещенных непосредственно на серверах VK, реализована через обходные решения. Возможна нестабильная работа данного модуля.
*   **Сторонние видеосервисы:** Интеграция видео с платформы YouTube функционирует штатно.
